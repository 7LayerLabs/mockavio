# Mockavio - Project Overview

## Vision Statement
Mockavio is a professional-grade AI visualization platform that helps realtors, designers, and restaurateurs transform client visions into reality through instant visual mockups and conceptual iterations.

## Brand Positioning

### PicForge (Consumer Brand)
- **Tagline:** "Break Reality. Make Them Weird, Epic, Yours."
- **Audience:** Creative individuals, hobbyists, content creators
- **Pricing:** $14/mo unlimited
- **Vibe:** Edgy, playful, experimental

### Mockavio (Professional Brand)
- **Tagline:** "Transform Client Visions Into Reality"
- **Audience:** Professional service providers (realtors, designers, restaurants)
- **Pricing:** $99-$299/mo + Enterprise
- **Vibe:** Professional, reliable, results-driven

## The Core Insight
The "creative journey" history sidebar in PicForge isn't just a nice feature - it's a client presentation tool. This is what makes Mockavio commercially valuable:
- Show clients 5+ variations instantly
- Walk through design iterations in real-time
- Present before/after sequences professionally
- Create visual proof-of-concept decks

## Technical Foundation
Mockavio is built on PicForge's proven technology:
- Same AI transformation engine (Gemini 2.5 Flash Image)
- Same codebase architecture (Next.js 16)
- Same authentication system (InstantDB)
- Same payment infrastructure (Stripe)
- Different branding, UX, and feature set

## Revenue Model
**B2B SaaS Subscription:**
- Starter: $99/mo (1 user, 500 images/mo)
- Professional: $299/mo (5 users, 2000 images/mo, API)
- Enterprise: Custom pricing (unlimited, white label)

**Target:** $10K MRR in 90 days, $50K MRR in 12 months

## Success Metrics
- Customer Acquisition Cost (CAC) < $300
- Lifetime Value (LTV) > $3,600 (12+ months retention)
- Monthly Churn < 5%
- Net Promoter Score (NPS) > 50
